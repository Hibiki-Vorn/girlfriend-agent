# openGirlfriend

This is my girlfriend. Her name is the nickname of my best friend of signal account.

This is my first artwork about AI which is depended on ollama and langchain.

This program is my girlfriend which can run shell scripts.

This can only run in the Linux or OSX.

**Please install ollama before run this program.**

Run following commands to start this program.

`python3 -m venv venv`

`source venv/bin/activate`

`pip install -r requirements.txt`

`python main.py`
